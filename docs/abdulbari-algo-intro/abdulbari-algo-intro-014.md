## 方法一：采样值比较法

![](img/7451c41815cd909302a987acf619eca6_28.png)

![](img/7451c41815cd909302a987acf619eca6_30.png)

![](img/7451c41815cd909302a987acf619eca6_32.png)

第一种比较方法是采样一些具体的输入值（n），计算两个函数在这些点上的值，然后进行比较。

![](img/7451c41815cd909302a987acf619eca6_36.png)

![](img/7451c41815cd909302a987acf619eca6_38.png)

![](img/7451c41815cd909302a987acf619eca6_40.png)

![](img/7451c41815cd909302a987acf619eca6_42.png)

以下是具体步骤：
1.  选择几个n值（例如2, 3, 4）。
2.  分别计算两个函数在这些n值下的结果。
3.  观察并总结大小关系。

![](img/7451c41815cd909302a987acf619eca6_44.png)

![](img/7451c41815cd909302a987acf619eca6_46.png)

![](img/7451c41815cd909302a987acf619eca6_47.png)

例如，比较 **f(n) = n²** 和 **g(n) = n³**：
*   当 n=2 时，f(2)=4，g(2)=8。4 < 8。
*   当 n=3 时，f(3)=9，g(3)=27。9 < 27。
*   当 n=4 时，f(4)=16，g(4)=64。16 < 64。

![](img/7451c41815cd909302a987acf619eca6_49.png)

![](img/7451c41815cd909302a987acf619eca6_50.png)

通过采样三个值，我们可以观察到 **n²** 始终小于 **n³**。因此，**n²** 是更小的函数。

![](img/7451c41815cd909302a987acf619eca6_52.png)

![](img/7451c41815cd909302a987acf619eca6_54.png)

这是一种通过采样具体数值来比较两个函数大小的方法。

## 方法二：对数比较法

![](img/7451c41815cd909302a987acf619eca6_65.png)

![](img/7451c41815cd909302a987acf619eca6_67.png)

![](img/7451c41815cd909302a987acf619eca6_69.png)

第二种方法是对等式两边同时取对数，然后比较对数结果的大小。这对于比较复杂的函数特别有用。

![](img/7451c41815cd909302a987acf619eca6_71.png)

![](img/7451c41815cd909302a987acf619eca6_73.png)

![](img/7451c41815cd909302a987acf619eca6_75.png)

![](img/7451c41815cd909302a987acf619eca6_76.png)

我们继续使用 **n²** 和 **n³** 的例子来演示这个方法：
1.  对两个函数取对数：`log(n²)` 和 `log(n³)`。
2.  应用对数公式进行化简：`log(n²) = 2 * log(n)`，`log(n³) = 3 * log(n)`。
3.  比较化简后的结果：`2 * log(n)` 和 `3 * log(n)`。显然，对于任意 n>1，`3 * log(n)` 都大于 `2 * log(n)`。

![](img/7451c41815cd909302a987acf619eca6_78.png)

![](img/7451c41815cd909302a987acf619eca6_80.png)

![](img/7451c41815cd909302a987acf619eca6_82.png)

因此，我们再次得出结论：**n³** 大于 **n²**。

![](img/7451c41815cd909302a987acf619eca6_84.png)

![](img/7451c41815cd909302a987acf619eca6_85.png)

## 必备对数公式

![](img/7451c41815cd909302a987acf619eca6_89.png)

![](img/7451c41815cd909302a987acf619eca6_90.png)

![](img/7451c41815cd909302a987acf619eca6_91.png)

![](img/7451c41815cd909302a987acf619eca6_92.png)

![](img/7451c41815cd909302a987acf619eca6_93.png)

![](img/7451c41815cd909302a987acf619eca6_95.png)

以下是进行比较时常用的对数公式：
*   `log(a * b) = log(a) + log(b)`
*   `log(a / b) = log(a) - log(b)`
*   `log(aᵇ) = b * log(a)`
*   换底公式：`logₐ(b) = logₓ(b) / logₓ(a)`
*   若 `aᵇ = n`，则 `b = logₐ(n)`

![](img/7451c41815cd909302a987acf619eca6_97.png)

![](img/7451c41815cd909302a987acf619eca6_99.png)

掌握这些公式将有助于我们解决涉及对数的比较问题。

**示例一：比较 f(n) = n² * log(n) 和 g(n) = n * (log n)¹⁰**

![](img/7451c41815cd909302a987acf619eca6_115.png)

![](img/7451c41815cd909302a987acf619eca6_116.png)

![](img/7451c41815cd909302a987acf619eca6_117.png)

仅凭观察难以判断。我们使用对数比较法：
1.  对两个函数取对数：
    *   `log(f(n)) = log(n² * log(n))`
    *   `log(g(n)) = log(n * (log n)¹⁰)`
2.  应用对数公式展开：
    *   `log(f(n)) = log(n²) + log(log(n)) = 2*log(n) + log(log(n))`
    *   `log(g(n)) = log(n) + log((log n)¹⁰) = log(n) + 10*log(log(n))`
3.  比较主要项：`2*log(n)` 与 `log(n)` 相比，前者更大；`log(log(n))` 与 `10*log(log(n))` 相比，后者更大。但 `log(n)` 的增长速度远快于 `log(log(n))`，因此起主导作用的是包含 `log(n)` 的项。由于 `2*log(n) > log(n)`，我们可以判断 `log(f(n)) > log(g(n))`，所以 **f(n) > g(n)**。

![](img/7451c41815cd909302a987acf619eca6_119.png)

![](img/7451c41815cd909302a987acf619eca6_121.png)

**示例二：比较 f(n) = 3^√n 和 g(n) = 2^(√n * log n)**

![](img/7451c41815cd909302a987acf619eca6_123.png)

![](img/7451c41815cd909302a987acf619eca6_125.png)

![](img/7451c41815cd909302a987acf619eca6_126.png)

这次我们不直接取对数，而是利用对数公式进行变换：
1.  注意到 `g(n) = 2^(√n * log n)`。根据公式 `a^(logₐ b) = b`，我们可以将 `n` 写作 `2^(log₂ n)`。但这需要更巧妙的变换。一个有效方法是直接比较 `log(f(n))` 和 `log(g(n))`：
    *   `log(f(n)) = log(3^√n) = √n * log 3`
    *   `log(g(n)) = log(2^(√n * log n)) = √n * log n * log 2 = √n * log n * 1` (因为log₂2=1)
2.  现在比较 `√n * log 3` 和 `√n * log n`。当 n 很大时，`log n` 会超过常数 `log 3`。因此，对于足够大的 n，有 `log(g(n)) > log(f(n))`，所以 **g(n) > f(n)**。

![](img/7451c41815cd909302a987acf619eca6_128.png)

![](img/7451c41815cd909302a987acf619eca6_130.png)

**示例三：理解“渐进相等”**

![](img/7451c41815cd909302a987acf619eca6_131.png)

![](img/7451c41815cd909302a987acf619eca6_132.png)

![](img/7451c41815cd909302a987acf619eca6_133.png)

![](img/7451c41815cd909302a987acf619eca6_135.png)

![](img/7451c41815cd909302a987acf619eca6_136.png)

考虑 `f(n) = 2n²` 和 `g(n) = n²`。谁更大？
*   从具体数值看，`2n²` 总是 `n²` 的两倍，所以 `2n²` 更大。
*   但从**渐进分析**（Asymptotic Analysis）的角度看，我们关注的是函数的增长**阶**（Order）。`2n²` 和 `n²` 的增长阶都是 **n²**。在讨论大O表示法（例如 O(n²)）时，常数系数（如2）被忽略，我们认为它们**渐进相等**。

## 总结

![](img/7451c41815cd909302a987acf619eca6_138.png)

![](img/7451c41815cd909302a987acf619eca6_139.png)

本节课我们一起学习了比较函数的两种核心方法：
1.  **采样值比较法**：通过计算函数在具体输入点的值进行直观比较，适用于简单或验证场景。
2.  **对数比较法**：对函数同时取对数并利用对数公式化简后再比较，是处理复杂函数比较的强有力工具。

![](img/7451c41815cd909302a987acf619eca6_141.png)

我们还回顾了关键的对数运算公式，并通过实例演示了如何运用这些方法判断函数的大小关系，以及理解了在渐进分析中“系数可忽略”的概念。掌握这些基础，对于后续分析算法复杂度至关重要。
