# 029：根函数递归关系求解教程 📚

![](img/f777a2682a3960b908ed2f36f7dbba82_1.png)

![](img/f777a2682a3960b908ed2f36f7dbba82_3.png)

在本节课中，我们将学习如何求解一种特殊的递归关系，其中递归调用的参数是输入值 `n` 的平方根。我们将通过一个具体的例子，使用回代法来逐步推导其时间复杂度。

![](img/f777a2682a3960b908ed2f36f7dbba82_5.png)

---

![](img/f777a2682a3960b908ed2f36f7dbba82_7.png)

上一节我们介绍了处理递减和分治递归关系的方法。本节中，我们来看看当递归调用涉及根函数（即参数为 `√n`）时，应如何求解。

![](img/f777a2682a3960b908ed2f36f7dbba82_9.png)

![](img/f777a2682a3960b908ed2f36f7dbba82_11.png)

首先，我们有一个算法，其递归关系如下。可以看到，函数通过传递 `n` 的平方根值来调用自身。

![](img/f777a2682a3960b908ed2f36f7dbba82_13.png)

**递归关系公式：**
```
T(n) = T(√n) + 1, 当 n > 2
```

这里需要注意，`n` 的取值必须大于 2。如果取 1 或 0，可能无法顺利求解此递归关系。因此，在处理根函数时，最小的 `n` 值应大于或等于 2。

现在，让我们开始求解这个递归式。我们直接使用回代法。

![](img/f777a2682a3960b908ed2f36f7dbba82_15.png)

![](img/f777a2682a3960b908ed2f36f7dbba82_16.png)

![](img/f777a2682a3960b908ed2f36f7dbba82_17.png)

![](img/f777a2682a3960b908ed2f36f7dbba82_18.png)

**第一步回代：**
将 `n` 替换为 `√n` 代入原式。
```
T(n) = T(n^(1/2)) + 1          // 方程 (1)
```

![](img/f777a2682a3960b908ed2f36f7dbba82_20.png)

![](img/f777a2682a3960b908ed2f36f7dbba82_21.png)

**第二步回代：**
将方程 (1) 中的 `T(n^(1/2))` 再次展开。
```
T(n) = [T((n^(1/2))^(1/2)) + 1] + 1 = T(n^(1/4)) + 2   // 方程 (2)
```

![](img/f777a2682a3960b908ed2f36f7dbba82_23.png)

![](img/f777a2682a3960b908ed2f36f7dbba82_25.png)

![](img/f777a2682a3960b908ed2f36f7dbba82_27.png)

**第三步回代：**
继续此过程。
```
T(n) = T(n^(1/8)) + 3          // 方程 (3)
```

![](img/f777a2682a3960b908ed2f36f7dbba82_29.png)

![](img/f777a2682a3960b908ed2f36f7dbba82_30.png)

![](img/f777a2682a3960b908ed2f36f7dbba82_32.png)

以下是经过 `k` 次回代后的通用形式：
```
T(n) = T(n^(1/(2^k))) + k
```

为了进一步求解，我们需要进行一个关键假设。我们假设 `n` 是 2 的幂次方，即 `n = 2^m`。这个假设在分析递归时很常见，能简化计算。

![](img/f777a2682a3960b908ed2f36f7dbba82_34.png)

![](img/f777a2682a3960b908ed2f36f7dbba82_35.png)

将 `n = 2^m` 代入通用形式：
```
T(2^m) = T(2^(m / 2^k)) + k
```

![](img/f777a2682a3960b908ed2f36f7dbba82_36.png)

![](img/f777a2682a3960b908ed2f36f7dbba82_37.png)

![](img/f777a2682a3960b908ed2f36f7dbba82_38.png)

![](img/f777a2682a3960b908ed2f36f7dbba82_39.png)

接下来，我们假设经过 `k` 次开方后，参数最终变为一个常数，这里我们取 `T(2)`。这意味着：
```
2^(m / 2^k) = 2^1
```

![](img/f777a2682a3960b908ed2f36f7dbba82_41.png)

![](img/f777a2682a3960b908ed2f36f7dbba82_43.png)

由此可得：
```
m / 2^k = 1
=> m = 2^k
=> k = log₂ m
```

![](img/f777a2682a3960b908ed2f36f7dbba82_45.png)

现在，我们需要将结果用原始的 `n` 来表示。因为 `n = 2^m`，所以 `m = log₂ n`。

![](img/f777a2682a3960b908ed2f36f7dbba82_47.png)

![](img/f777a2682a3960b908ed2f36f7dbba82_49.png)

将 `m` 代入 `k` 的表达式：
```
k = log₂ m = log₂ (log₂ n)
```

![](img/f777a2682a3960b908ed2f36f7dbba82_51.png)

因此，我们得到递归式的解为 `Θ(log log n)`，即以 2 为底的对数的对数。

![](img/f777a2682a3960b908ed2f36f7dbba82_52.png)

![](img/f777a2682a3960b908ed2f36f7dbba82_54.png)

本节课中我们一起学习了如何求解参数为根函数的递归关系。我们使用了回代法，并借助 `n` 是 2 的幂次方这一假设，最终推导出其时间复杂度为 **`Θ(log log n)`**。这种方法也适用于主定理中处理类似分治函数的情况。
