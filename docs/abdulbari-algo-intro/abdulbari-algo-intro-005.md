# 005：频率计数法 📊

在本节课中，我们将学习一种分析算法时间复杂度的核心方法——**频率计数法**。我们将通过分析几个具体的算法示例，来掌握如何应用此方法计算其时间与空间复杂度。

![](img/6354015c9db7fb13cf68bd01ae8855a3_5.png)

![](img/6354015c9db7fb13cf68bd01ae8855a3_6.png)

## 概述：什么是频率计数法？

![](img/6354015c9db7fb13cf68bd01ae8855a3_8.png)

频率计数法的核心思想是：**为算法中的每一条语句分配一个单位时间**。如果某条语句重复执行多次，则计算其执行的**频率**。将所有语句的执行时间（频率）相加，即可得到该算法的总时间函数。

![](img/6354015c9db7fb13cf68bd01ae8855a3_9.png)

上一节我们介绍了算法分析的重要性，本节中我们来看看如何具体地为一个算法“计数”。

---

## 示例一：数组求和算法

![](img/6354015c9db7fb13cf68bd01ae8855a3_11.png)

首先，我们分析一个计算数组所有元素之和的简单算法。假设数组 `A` 的大小为 `n`。

![](img/6354015c9db7fb13cf68bd01ae8855a3_13.png)

以下是该算法的核心代码逻辑：
```c
sum = 0;
for (i = 0; i < n; i++) {
    sum = sum + A[i];
}
```

![](img/6354015c9db7fb13cf68bd01ae8855a3_15.png)

![](img/6354015c9db7fb13cf68bd01ae8855a3_17.png)

### 逐步时间分析

我们使用频率计数法为每条语句计算执行次数。

1.  **`sum = 0;`**
    *   此语句执行**1次**。

![](img/6354015c9db7fb13cf68bd01ae8855a3_19.png)

![](img/6354015c9db7fb13cf68bd01ae8855a3_20.png)

2.  **`for (i = 0; i < n; i++)`** 中的循环条件 `i < n`
    *   此条件会被检查 `n + 1` 次。当 `i` 从 `0` 到 `n-1` 时，条件为真，共 `n` 次。当 `i` 等于 `n` 时，条件为假，检查第 `n+1` 次后循环停止。

![](img/6354015c9db7fb13cf68bd01ae8855a3_22.png)

3.  **循环体内的 `sum = sum + A[i];`**
    *   此语句在循环内，会执行 **`n` 次**。

![](img/6354015c9db7fb13cf68bd01ae8855a3_24.png)

![](img/6354015c9db7fb13cf68bd01ae8855a3_26.png)

4.  **最后的输出或返回语句**
    *   我们假设有一条输出 `sum` 的语句，它执行**1次**。

![](img/6354015c9db7fb13cf68bd01ae8855a3_27.png)

### 计算时间函数

![](img/6354015c9db7fb13cf68bd01ae8855a3_29.png)

将以上所有语句的执行次数（频率）相加，我们得到时间函数 **`f(n)`**：
```
f(n) = 1 + (n+1) + n + 1 = 2n + 3
```

![](img/6354015c9db7fb13cf68bd01ae8855a3_31.png)

### 确定时间复杂度

![](img/6354015c9db7fb13cf68bd01ae8855a3_33.png)

我们关注当 `n` 很大时，函数中增长最快的项（最高次项）。`2n + 3` 的最高次项是 `n`，其阶数为 1。
因此，该算法的**时间复杂度为 `O(n)`**。

![](img/6354015c9db7fb13cf68bd01ae8855a3_35.png)

### 分析空间复杂度

![](img/6354015c9db7fb13cf68bd01ae8855a3_37.png)

空间复杂度分析算法运行所需的额外内存空间。

![](img/6354015c9db7fb13cf68bd01ae8855a3_39.png)

![](img/6354015c9db7fb13cf68bd01ae8855a3_41.png)

以下是该算法使用的变量：
*   `A`: 一个大小为 `n` 的数组。
*   `sum`: 一个存储结果的变量。
*   `i`: 一个循环计数器。

![](img/6354015c9db7fb13cf68bd01ae8855a3_43.png)

假设每个变量占用1个单位空间（`sum` 和 `i` 各占1，数组 `A` 占 `n`），则总空间函数为 `n + 2`。
其最高次项同样是 `n`，因此**空间复杂度也为 `O(n)`**。

![](img/6354015c9db7fb13cf68bd01ae8855a3_45.png)

![](img/6354015c9db7fb13cf68bd01ae8855a3_47.png)

![](img/6354015c9db7fb13cf68bd01ae8855a3_49.png)

## 示例二：矩阵加法算法

![](img/6354015c9db7fb13cf68bd01ae8855a3_51.png)

接下来，我们分析一个更复杂的算法：将两个 `n x n` 的矩阵 `A` 和 `B` 相加，结果存入矩阵 `C`。

![](img/6354015c9db7fb13cf68bd01ae8855a3_53.png)

算法使用两层嵌套循环：
```c
for (i = 0; i < n; i++) {
    for (j = 0; j < n; j++) {
        C[i][j] = A[i][j] + B[i][j];
    }
}
```

![](img/6354015c9db7fb13cf68bd01ae8855a3_55.png)

![](img/6354015c9db7fb13cf68bd01ae8855a3_56.png)

### 逐步时间分析

1.  **外层循环 `for (i = 0; i < n; i++)` 的条件**
    *   执行 **`n + 1`** 次。

![](img/6354015c9db7fb13cf68bd01ae8855a3_58.png)

![](img/6354015c9db7fb13cf68bd01ae8855a3_60.png)

2.  **内层循环 `for (j = 0; j < n; j++)` 的条件**
    *   对于外层循环的每一次迭代，内层循环条件都会执行 `n + 1` 次。由于外层循环执行 `n` 次，所以该条件总共执行 **`n * (n + 1)`** 次。

![](img/6354015c9db7fb13cf68bd01ae8855a3_62.png)

3.  **最内层的赋值语句 `C[i][j] = A[i][j] + B[i][j];`**
    *   该语句位于两层循环内部，总共执行 **`n * n = n²`** 次。

![](img/6354015c9db7fb13cf68bd01ae8855a3_64.png)

![](img/6354015c9db7fb13cf68bd01ae8855a3_66.png)

### 计算时间函数

![](img/6354015c9db7fb13cf68bd01ae8855a3_68.png)

简化并合并所有执行次数，我们得到主要的时间函数项：
```
f(n) ≈ n² + ... (其他低阶项)
```
其中，增长最快的项是 `n²`。

![](img/6354015c9db7fb13cf68bd01ae8855a3_70.png)

### 确定时间复杂度

![](img/6354015c9db7fb13cf68bd01ae8855a3_72.png)

因此，该矩阵加法算法的**时间复杂度为 `O(n²)`**。

![](img/6354015c9db7fb13cf68bd01ae8855a3_74.png)

![](img/6354015c9db7fb13cf68bd01ae8855a3_76.png)

### 分析空间复杂度

![](img/6354015c9db7fb13cf68bd01ae8855a3_78.png)

![](img/6354015c9db7fb13cf68bd01ae8855a3_80.png)

![](img/6354015c9db7fb13cf68bd01ae8855a3_82.png)

以下是该算法使用的变量：
*   `A`, `B`, `C`: 三个 `n x n` 的二维数组，各占 `n²` 空间。
*   `i`, `j`: 两个循环计数器，各占1空间。

![](img/6354015c9db7fb13cf68bd01ae8855a3_84.png)

![](img/6354015c9db7fb13cf68bd01ae8855a3_86.png)

总空间函数约为 `3n² + 2`。其最高次项是 `n²`，因此**空间复杂度也为 `O(n²)`**。

![](img/6354015c9db7fb13cf68bd01ae8855a3_88.png)

![](img/6354015c9db7fb13cf68bd01ae8855a3_90.png)

## 示例三：矩阵乘法算法

最后，我们分析一个三层嵌套循环的算法：计算两个 `n x n` 矩阵 `A` 和 `B` 的乘积，结果存入矩阵 `C`。

算法核心结构如下：
```c
for (i = 0; i < n; i++) {
    for (j = 0; j < n; j++) {
        C[i][j] = 0;
        for (k = 0; k < n; k++) {
            C[i][j] = C[i][j] + A[i][k] * B[k][j];
        }
    }
}
```

![](img/6354015c9db7fb13cf68bd01ae8855a3_94.png)

### 逐步时间分析

![](img/6354015c9db7fb13cf68bd01ae8855a3_96.png)

![](img/6354015c9db7fb13cf68bd01ae8855a3_97.png)

1.  **最内层循环 `for (k = 0; k < n; k++)` 及其内部语句**
    *   该循环条件执行 `n + 1` 次，内部赋值语句执行 `n` 次。但这对每一对 `(i, j)` 都会发生。

![](img/6354015c9db7fb13cf68bd01ae8855a3_99.png)

2.  **中层和内层循环**
    *   对于外层循环的每个 `i`，中层循环运行 `n` 次。
    *   对于中层的每个 `j`，最内层循环运行 `n` 次。
    *   因此，最内层的核心乘法语句 `C[i][j] = C[i][j] + A[i][k] * B[k][j];` 总共执行 **`n * n * n = n³`** 次。

![](img/6354015c9db7fb13cf68bd01ae8855a3_100.png)

![](img/6354015c9db7fb13cf68bd01ae8855a3_102.png)

### 计算时间函数

![](img/6354015c9db7fb13cf68bd01ae8855a3_104.png)

忽略低阶项和常数系数，时间函数主要由 `n³` 项主导。
```
f(n) ≈ 2n³ + 3n² + 2n + 1
```

![](img/6354015c9db7fb13cf68bd01ae8855a3_106.png)

### 确定时间复杂度

![](img/6354015c9db7fb13cf68bd01ae8855a3_108.png)

最高次项是 `n³`，因此该矩阵乘法算法的**时间复杂度为 `O(n³)`**。

![](img/6354015c9db7fb13cf68bd01ae8855a3_110.png)

![](img/6354015c9db7fb13cf68bd01ae8855a3_112.png)

![](img/6354015c9db7fb13cf68bd01ae8855a3_113.png)

### 分析空间复杂度

![](img/6354015c9db7fb13cf68bd01ae8855a3_115.png)

![](img/6354015c9db7fb13cf68bd01ae8855a3_116.png)

算法使用的变量：
*   `A`, `B`, `C`: 三个 `n x n` 矩阵，占 `3n²` 空间。
*   `i`, `j`, `k`: 三个循环计数器，占3空间。

![](img/6354015c9db7fb13cf68bd01ae8855a3_118.png)

![](img/6354015c9db7fb13cf68bd01ae8855a3_120.png)

总空间函数约为 `3n² + 3`。其最高次项是 `n²`，因此**空间复杂度为 `O(n²)`**。注意，虽然时间开销是立方级，但空间开销仍是平方级。

![](img/6354015c9db7fb13cf68bd01ae8855a3_122.png)

![](img/6354015c9db7fb13cf68bd01ae8855a3_123.png)

## 总结 📝

![](img/6354015c9db7fb13cf68bd01ae8855a3_129.png)

本节课中我们一起学习了**频率计数法**，这是一种通过统计语句执行次数来分析算法效率的基础方法。

![](img/6354015c9db7fb13cf68bd01ae8855a3_131.png)

我们通过三个由简到繁的例子实践了该方法：
1.  **数组求和**：时间复杂度 `O(n)`，空间复杂度 `O(n)`。
2.  **矩阵加法**：时间复杂度 `O(n²)`，空间复杂度 `O(n²)`。
3.  **矩阵乘法**：时间复杂度 `O(n³)`，空间复杂度 `O(n²)`。

![](img/6354015c9db7fb13cf68bd01ae8855a3_133.png)

关键要点在于：
*   为每条语句分配单位时间并计算其执行频率。
*   将频率相加得到时间函数 `f(n)`。
*   取 `f(n)` 中最高次项（忽略系数和低阶项）来确定算法的**时间复杂度**。
*   类似地，统计算法使用的所有变量所占空间来确定**空间复杂度**。

![](img/6354015c9db7fb13cf68bd01ae8855a3_135.png)

掌握这一方法，你就能对许多简单算法的性能进行量化分析和比较了。在接下来的课程中，我们将看到更多样的算法及其分析方法。
