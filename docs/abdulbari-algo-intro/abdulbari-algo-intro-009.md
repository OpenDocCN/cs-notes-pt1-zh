# 009：函数复杂度类别 🧮

在本节课中，我们将学习算法分析中常见的几种时间复杂度函数类别。理解这些类别有助于我们快速判断和比较不同算法的效率。

![](img/0cbfd083f6c5444b617a3eb93d407213_1.png)

![](img/0cbfd083f6c5444b617a3eb93d407213_2.png)

上一节我们介绍了如何用大O记法表示算法的时间复杂度。本节中，我们来看看这些时间复杂度具体对应哪些类型的函数。

![](img/0cbfd083f6c5444b617a3eb93d407213_4.png)

## 常数时间

首先，我们得到的时间复杂度可能是 **O(1)**。

```math
O(1)
```

![](img/0cbfd083f6c5444b617a3eb93d407213_6.png)

![](img/0cbfd083f6c5444b617a3eb93d407213_7.png)

这意味着算法的运行时间是一个常数。无论输入规模 `n` 如何变化，执行时间都保持不变。例如，时间复杂度是 1、5 或 5000，都属于常数时间。

**核心概念**：算法的运行时间不依赖于输入规模 `n`。

![](img/0cbfd083f6c5444b617a3eb93d407213_9.png)

## 对数时间

![](img/0cbfd083f6c5444b617a3eb93d407213_11.png)

接下来，我们得到的时间复杂度可能是 **O(log n)**。

![](img/0cbfd083f6c5444b617a3eb93d407213_13.png)

```math
O(\log n)
```

![](img/0cbfd083f6c5444b617a3eb93d407213_15.png)

这被称为对数时间复杂度。无论对数的底数是 2、3 还是其他值，我们都将其归类为对数级。

![](img/0cbfd083f6c5444b617a3eb93d407213_17.png)

## 线性时间

然后，我们得到的时间复杂度可能是 **O(n)**。

![](img/0cbfd083f6c5444b617a3eb93d407213_19.png)

```math
O(n)
```

![](img/0cbfd083f6c5444b617a3eb93d407213_20.png)

![](img/0cbfd083f6c5444b617a3eb93d407213_21.png)

这被称为线性时间复杂度。即使函数的具体形式是 `f(n) = 2n + 3`、`f(n) = 500n + 700` 或 `f(n) = n/5000 + 6`，其时间复杂度仍然是 **O(n)**。

**核心概念**：我们关注的是多项式函数的最高次项。当最高次项是 `n` 的一次方时，我们就写作 **O(n)**，而不写出具体的系数或低阶项。

![](img/0cbfd083f6c5444b617a3eb93d407213_23.png)

## 平方时间与立方时间

![](img/0cbfd083f6c5444b617a3eb93d407213_25.png)

对于某些算法，我们得到的时间复杂度可能是 **O(n²)**。

![](img/0cbfd083f6c5444b617a3eb93d407213_27.png)

```math
O(n^2)
```

![](img/0cbfd083f6c5444b617a3eb93d407213_29.png)

这被称为平方时间复杂度。例如，矩阵乘法算法的时间复杂度就是 **O(n³)**，即立方时间复杂度。

## 指数时间

![](img/0cbfd083f6c5444b617a3eb93d407213_31.png)

![](img/0cbfd083f6c5444b617a3eb93d407213_33.png)

对于另一些算法，你可能会得到像 **O(2ⁿ)** 这样的时间复杂度。

```math
O(2^n)
```

这被称为指数时间复杂度。它可能是 `2ⁿ`、`3ⁿ` 或 `nⁿ` 等形式。我们目前尚未详细讨论这类算法，但在后续课程中可能会遇到。

![](img/0cbfd083f6c5444b617a3eb93d407213_35.png)

## 函数类别总结

综上所述，所有的时间复杂度函数都可以归入不同的类别或“函数类”。

以下是常见的时间复杂度类别，从最优到最差排列：

*   **常数类**：`O(1)`
*   **对数类**：`O(log n)`
*   **线性类**：`O(n)`
*   **平方类**：`O(n²)`
*   **立方类**：`O(n³)`
*   **指数类**：`O(2ⁿ)`

![](img/0cbfd083f6c5444b617a3eb93d407213_37.png)

![](img/0cbfd083f6c5444b617a3eb93d407213_38.png)

本节课中我们一起学习了算法时间复杂度的主要类别，包括常数时间、对数时间、线性时间、平方时间、立方时间和指数时间。理解这些类别是分析和比较算法效率的基础。
