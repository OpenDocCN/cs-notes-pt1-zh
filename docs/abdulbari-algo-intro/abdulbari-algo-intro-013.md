## 一般性质

上一节我们介绍了渐近符号的基本概念，本节中我们来看看它们的一些通用性质。

![](img/5822a9e0251b581b40ef56260d87bc92_23.png)

![](img/5822a9e0251b581b40ef56260d87bc92_25.png)

### 常数乘法性质

![](img/5822a9e0251b581b40ef56260d87bc92_27.png)

![](img/5822a9e0251b581b40ef56260d87bc92_28.png)

![](img/5822a9e0251b581b40ef56260d87bc92_29.png)

如果一个函数 `f(n)` 是 `O(g(n))`，那么将该函数乘以一个正常数 `c` 后，其渐近复杂度不变。

**公式**：如果 `f(n) = O(g(n))`，那么 `c * f(n) = O(g(n))`，其中 `c > 0`。

![](img/5822a9e0251b581b40ef56260d87bc92_31.png)

![](img/5822a9e0251b581b40ef56260d87bc92_32.png)

![](img/5822a9e0251b581b40ef56260d87bc92_34.png)

例如，假设 `f(n) = 2n² + 5`，并且已知 `f(n) = O(n²)`。那么 `7 * f(n) = 7*(2n²+5) = 14n² + 35` 也同样是 `O(n²)`。因为函数的阶数（最高次项）没有改变，所以其渐近复杂度不变。

![](img/5822a9e0251b581b40ef56260d87bc92_36.png)

![](img/5822a9e0251b581b40ef56260d87bc92_38.png)

![](img/5822a9e0251b581b40ef56260d87bc92_39.png)

这个性质不仅适用于大O符号，也适用于Omega和Theta符号。

![](img/5822a9e0251b581b40ef56260d87bc92_41.png)

![](img/5822a9e0251b581b40ef56260d87bc92_43.png)

![](img/5822a9e0251b581b40ef56260d87bc92_45.png)

**公式**：
* 如果 `f(n) = Ω(g(n))`，那么 `c * f(n) = Ω(g(n))`。
* 如果 `f(n) = Θ(g(n))`，那么 `c * f(n) = Θ(g(n))`。

![](img/5822a9e0251b581b40ef56260d87bc92_47.png)

![](img/5822a9e0251b581b40ef56260d87bc92_49.png)

![](img/5822a9e0251b581b40ef56260d87bc92_50.png)

### 自反性

一个函数是其自身的上界（对于大O符号）和下界（对于Omega符号）。

![](img/5822a9e0251b581b40ef56260d87bc92_52.png)

**公式**：`f(n) = O(f(n))` 且 `f(n) = Ω(f(n))`。

![](img/5822a9e0251b581b40ef56260d87bc92_54.png)

![](img/5822a9e0251b581b40ef56260d87bc92_56.png)

这意味着，对于任意函数 `f(n)`，它本身就是一个有效的上界（大于或等于自身）和下界（小于或等于自身）。例如，`n² = O(n²)` 是成立的。

![](img/5822a9e0251b581b40ef56260d87bc92_58.png)

![](img/5822a9e0251b581b40ef56260d87bc92_60.png)

### 传递性

![](img/5822a9e0251b581b40ef56260d87bc92_62.png)

![](img/5822a9e0251b581b40ef56260d87bc92_64.png)

![](img/5822a9e0251b581b40ef56260d87bc92_66.png)

传递性是指，如果函数 `f` 的上界是 `g`，而 `g` 的上界是 `h`，那么 `h` 也是 `f` 的上界。

![](img/5822a9e0251b581b40ef56260d87bc92_68.png)

![](img/5822a9e0251b581b40ef56260d87bc92_70.png)

![](img/5822a9e0251b581b40ef56260d87bc92_72.png)

**公式**：如果 `f(n) = O(g(n))` 且 `g(n) = O(h(n))`，那么 `f(n) = O(h(n))`。

![](img/5822a9e0251b581b40ef56260d87bc92_74.png)

![](img/5822a9e0251b581b40ef56260d87bc92_76.png)

![](img/5822a9e0251b581b40ef56260d87bc92_78.png)

让我们通过一个例子来理解：
* 设 `f(n) = n`
* 设 `g(n) = n²`
* 设 `h(n) = n³`

![](img/5822a9e0251b581b40ef56260d87bc92_80.png)

![](img/5822a9e0251b581b40ef56260d87bc92_82.png)

我们知道 `n = O(n²)` 且 `n² = O(n³)`。根据传递性，可以得出 `n = O(n³)`。这个性质同样适用于Omega和Theta符号。

### 对称性

![](img/5822a9e0251b581b40ef56260d87bc92_84.png)

![](img/5822a9e0251b581b40ef56260d87bc92_86.png)

![](img/5822a9e0251b581b40ef56260d87bc92_88.png)

对称性特指Theta符号。如果两个函数互为Theta关系，那么这种关系是对称的。

![](img/5822a9e0251b581b40ef56260d87bc92_90.png)

![](img/5822a9e0251b581b40ef56260d87bc92_92.png)

![](img/5822a9e0251b581b40ef56260d87bc92_94.png)

**公式**：如果 `f(n) = Θ(g(n))`，那么 `g(n) = Θ(f(n))`。

![](img/5822a9e0251b581b40ef56260d87bc92_96.png)

![](img/5822a9e0251b581b40ef56260d87bc92_98.png)

![](img/5822a9e0251b581b40ef56260d87bc92_100.png)

例如，如果 `f(n) = n²` 且 `g(n) = n²`，那么 `f(n) = Θ(n²)` 且 `g(n) = Θ(n²)`，两者互为Theta关系。这个性质在大O和Omega符号中不成立，它们具有的是转置对称性。

### 转置对称性

![](img/5822a9e0251b581b40ef56260d87bc92_102.png)

转置对称性描述了大O和Omega符号之间的关系：如果一个函数是另一个函数的上界，那么反过来，后者就是前者的下界。

![](img/5822a9e0251b581b40ef56260d87bc92_103.png)

![](img/5822a9e0251b581b40ef56260d87bc92_104.png)

![](img/5822a9e0251b581b40ef56260d87bc92_106.png)

![](img/5822a9e0251b581b40ef56260d87bc92_108.png)

**公式**：如果 `f(n) = O(g(n))`，那么 `g(n) = Ω(f(n))`。

![](img/5822a9e0251b581b40ef56260d87bc92_110.png)

![](img/5822a9e0251b581b40ef56260d87bc92_112.png)

![](img/5822a9e0251b581b40ef56260d87bc92_114.png)

例如：
* 设 `f(n) = n`
* 设 `g(n) = n²`

因为 `n = O(n²)`（n是n²的上界），所以反过来，`n² = Ω(n)`（n²是n的下界）。

![](img/5822a9e0251b581b40ef56260d87bc92_116.png)

![](img/5822a9e0251b581b40ef56260d87bc92_117.png)

![](img/5822a9e0251b581b40ef56260d87bc92_118.png)

### 夹逼定理与Theta关系

![](img/5822a9e0251b581b40ef56260d87bc92_120.png)

如果一个函数同时被同一个函数 `g(n)` 从上方（大O）和下方（Omega）界定，那么该函数就是 `g(n)` 的Theta。

![](img/5822a9e0251b581b40ef56260d87bc92_122.png)

**公式**：如果 `f(n) = O(g(n))` 且 `f(n) = Ω(g(n))`，那么 `f(n) = Θ(g(n))`。

![](img/5822a9e0251b581b40ef56260d87bc92_124.png)

![](img/5822a9e0251b581b40ef56260d87bc92_125.png)

![](img/5822a9e0251b581b40ef56260d87bc92_127.png)

这直观地说明了Theta符号的含义：它描述了两个函数增长速率“相等”的情况，即一个函数被另一个函数紧密地夹在中间。

![](img/5822a9e0251b581b40ef56260d87bc92_129.png)

### 函数相加的性质

![](img/5822a9e0251b581b40ef56260d87bc92_130.png)

![](img/5822a9e0251b581b40ef56260d87bc92_132.png)

以下是两个函数相加时的渐近复杂度规则。

![](img/5822a9e0251b581b40ef56260d87bc92_134.png)

如果已知 `f(n) = O(g(n))` 且 `d(n) = O(e(n))`，那么两个函数之和的复杂度由其中增长更快的项决定。

![](img/5822a9e0251b581b40ef56260d87bc92_135.png)

![](img/5822a9e0251b581b40ef56260d87bc92_136.png)

![](img/5822a9e0251b581b40ef56260d87bc92_138.png)

![](img/5822a9e0251b581b40ef56260d87bc92_140.png)

**公式**：`f(n) + d(n) = O( max( g(n), e(n) ) )`

![](img/5822a9e0251b581b40ef56260d87bc92_142.png)

![](img/5822a9e0251b581b40ef56260d87bc92_144.png)

让我们看一个例子：
* 设 `f(n) = n`，则 `f(n) = O(n)`
* 设 `d(n) = n²`，则 `d(n) = O(n²)`
* 那么 `f(n) + d(n) = n + n²`

由于 `n²` 是增长更快的项，所以 `n + n² = O(n²)`，即 `O( max(n, n²) )`。

![](img/5822a9e0251b581b40ef56260d87bc92_146.png)

![](img/5822a9e0251b581b40ef56260d87bc92_148.png)

### 函数相乘的性质

![](img/5822a9e0251b581b40ef56260d87bc92_149.png)

![](img/5822a9e0251b581b40ef56260d87bc92_151.png)

以下是两个函数相乘时的渐近复杂度规则。

![](img/5822a9e0251b581b40ef56260d87bc92_153.png)

如果 `f(n) = O(g(n))` 且 `d(n) = O(e(n))`，那么两个函数之积的复杂度是它们各自复杂度的乘积。

![](img/5822a9e0251b581b40ef56260d87bc92_155.png)

![](img/5822a9e0251b581b40ef56260d87bc92_156.png)

![](img/5822a9e0251b581b40ef56260d87bc92_157.png)

**公式**：`f(n) * d(n) = O( g(n) * e(n) )`

![](img/5822a9e0251b581b40ef56260d87bc92_159.png)

例如：
* 设 `f(n) = n`，则 `f(n) = O(n)`
* 设 `d(n) = n²`，则 `d(n) = O(n²)`
* 那么 `f(n) * d(n) = n * n² = n³`
* 根据性质，`n * n² = O(n * n²) = O(n³)`，与直接计算结果一致。

![](img/5822a9e0251b581b40ef56260d87bc92_161.png)

![](img/5822a9e0251b581b40ef56260d87bc92_163.png)

## 总结 🎯

![](img/5822a9e0251b581b40ef56260d87bc92_165.png)

![](img/5822a9e0251b581b40ef56260d87bc92_166.png)

本节课中我们一起学习了渐近符号的七个核心性质：
1.  **常数乘法性质**：乘以常数不改变渐近阶。
2.  **自反性**：函数是其自身的上界和下界。
3.  **传递性**：上界关系可以传递。
4.  **对称性**：Theta关系是对称的。
5.  **转置对称性**：大O和Omega符号互为转置。
6.  **夹逼定理**：被同一函数同时从上和下界定，则构成Theta关系。
7.  **和与积的性质**：和的复杂度取最大项，积的复杂度为复杂度相乘。

![](img/5822a9e0251b581b40ef56260d87bc92_167.png)

![](img/5822a9e0251b581b40ef56260d87bc92_168.png)

![](img/5822a9e0251b581b40ef56260d87bc92_170.png)

掌握这些性质是进行严谨算法复杂度分析的基础，它们帮助我们简化比较过程并得出可靠的结论。
