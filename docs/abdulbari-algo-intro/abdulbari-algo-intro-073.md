## 什么是团？🔍

![](img/d5d0d201bd565083e54ced01d7007158_7.png)

![](img/d5d0d201bd565083e54ced01d7007158_8.png)

上一节我们介绍了课程目标，本节中我们来看看第一个核心概念：团。

![](img/d5d0d201bd565083e54ced01d7007158_10.png)

首先，我们需要理解什么是完全图。如果一个图有 `n` 个顶点，并且每个顶点都与其他所有顶点相连，那么这个图就是一个完全图。一个具有 `n` 个顶点的完全图，其边的数量为 `n * (n-1) / 2`。

![](img/d5d0d201bd565083e54ced01d7007158_12.png)

![](img/d5d0d201bd565083e54ced01d7007158_13.png)

**公式：** 完全图的边数 = `n * (n-1) / 2`

现在，我们来看团的定义。团是一个图的子图，并且这个子图本身是一个完全图。换句话说，团是原图中一个顶点集合，这个集合中任意两个顶点之间都有边相连。

## 什么是NP难问题？🤔

![](img/d5d0d201bd565083e54ced01d7007158_34.png)

![](img/d5d0d201bd565083e54ced01d7007158_36.png)

我们已经明确了要研究的问题类型。那么，为什么说团判定问题是一个“NP难”的图问题呢？

简单来说，NP难问题是指那些至少和NP中最难的问题一样难的问题。目前没有已知的多项式时间算法可以解决所有NP难问题的实例。如果一个判定问题是NP难的，那么其对应的优化问题通常也是NP难的。

![](img/d5d0d201bd565083e54ced01d7007158_38.png)

## 如何证明一个问题是NP难的？🔨

![](img/d5d0d201bd565083e54ced01d7007158_40.png)

![](img/d5d0d201bd565083e54ced01d7007158_42.png)

要证明一个新问题（记作 L2）是NP难的，我们需要使用“规约”技术。以下是证明步骤：

1.  选择一个已知的NP难问题（记作 L1）。
2.  展示如何将 L1 的任意实例，在多项式时间内，转化为 L2 的一个实例。
3.  证明这个转化是有效的：即 L1 实例的答案为“是”，当且仅当对应的 L2 实例的答案也为“是”。

![](img/d5d0d201bd565083e54ced01d7007158_44.png)

![](img/d5d0d201bd565083e54ced01d7007158_46.png)

![](img/d5d0d201bd565083e54ced01d7007158_47.png)

这意味着，如果我们能在多项式时间内解决 L2，那么我们也能在多项式时间内解决 L1。由于 L1 是NP难的（公认没有多项式解法），因此 L2 也必须是NP难的。

![](img/d5d0d201bd565083e54ced01d7007158_49.png)

我们将使用已知的NP难问题——**布尔可满足性问题**（SAT问题）——来证明团判定问题是NP难的。

![](img/d5d0d201bd565083e54ced01d7007158_51.png)

## 证明：SAT问题可规约到团判定问题 🧠

![](img/d5d0d201bd565083e54ced01d7007158_53.png)

现在，让我们进入核心环节，证明SAT问题可以规约到团判定问题，从而证明后者是NP难的。

我们将从一个具体的SAT公式例子开始，展示如何构造一个对应的图。

**给定一个SAT公式：**
假设我们有一个合取范式公式 F，包含三个变量（x1, x2, x3）和三个子句（K=3）：
`F = (x1 ∨ x2) ∧ (¬x1 ∨ ¬x2) ∧ (x1 ∨ x3)`

![](img/d5d0d201bd565083e54ced01d7007158_55.png)

**构造对应的图 G：**
以下是构造规则：
1.  **顶点**：为公式 F 中每个子句里的每个**文字**（变量或其否定）创建一个顶点。顶点标记为 `(文字, 子句号)`。在我们的例子中，顶点为：`(x1,1)`, `(x2,1)`, `(¬x1,2)`, `(¬x2,2)`, `(x1,3)`, `(x3,3)`。
2.  **边**：连接不同子句的顶点，但有一个例外。
    *   **规则**：连接顶点 `(a, i)` 和 `(b, j)`，当且仅当满足以下两个条件：
        *   `i ≠ j` （顶点来自不同的子句）。
        *   `b ≠ ¬a` （两个文字不互为否定）。

![](img/d5d0d201bd565083e54ced01d7007158_57.png)

![](img/d5d0d201bd565083e54ced01d7007158_59.png)

根据以上规则，我们可以画出图 G。可以验证，在这个构造出的图 G 中，存在一个大小为 3（等于子句数 K）的团，例如顶点 `{(x2,1), (¬x1,2), (x3,3)}`。

**为什么这能证明规约有效？**
*   **如果 F 可满足**：则每个子句中至少有一个文字为真。从每个子句中选取一个为真的文字对应的顶点，这些顶点之间必然有边相连（因为它们来自不同子句且不互为否定），从而构成一个大小为 K 的团。
*   **如果 G 有一个大小为 K 的团**：因为这个团中任意两个顶点都相连，根据边的构造规则，它们必然来自不同的子句且代表的文字不互为否定。因此，我们可以将团中顶点对应的文字赋值为“真”，这个赋值会使每个子句为真，从而使整个公式 F 为真。

![](img/d5d0d201bd565083e54ced01d7007158_61.png)

![](img/d5d0d201bd565083e54ced01d7007158_62.png)

![](img/d5d0d201bd565083e54ced01d7007158_63.png)

![](img/d5d0d201bd565083e54ced01d7007158_65.png)

![](img/d5d0d201bd565083e54ced01d7007158_66.png)

![](img/d5d0d201bd565083e54ced01d7007158_68.png)

我们成功地将一个SAT问题实例转化为了一个团判定问题实例，并且两个问题的答案一致性得到了保证。这个转化过程（构造图）显然可以在多项式时间内完成。因此，SAT问题可以规约到团判定问题。

![](img/d5d0d201bd565083e54ced01d7007158_70.png)

## 总结 📚

![](img/d5d0d201bd565083e54ced01d7007158_72.png)

本节课中我们一起学习了：
1.  **团**的定义：一个图中完全的子图。
2.  **判定问题**（回答是/否）与**优化问题**（寻找最优值）的区别。
3.  **NP难问题**的基本概念：目前没有已知高效通用解法的问题类别。
4.  证明问题为NP难的通用方法：通过**多项式时间规约**从一个已知NP难问题转化而来。
5.  团判定问题是NP难的证明：我们演示了如何将 **布尔可满足性问题（SAT）** 的实例，在多项式时间内规约为**团判定问题**的实例，从而证明了后者是NP难的。

![](img/d5d0d201bd565083e54ced01d7007158_74.png)

要证明团判定问题是NP完全的，还需要证明它本身也在NP类中（即存在一个非确定性图灵机可以在多项式时间内验证其解），这通常通过构造一个验证算法来完成。但可以确定的是，团判定问题是一个经典的NP难（也是NP完全）问题。
