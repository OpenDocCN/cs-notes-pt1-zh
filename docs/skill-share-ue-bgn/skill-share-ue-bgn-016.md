# 016：室内照明 💡

在本节课中，我们将学习如何为室内场景设置照明。我们将从阻止外部光线开始，然后学习如何放置和配置不同类型的灯光，包括聚光灯和矩形灯。最后，我们还会创建一个能自发光的材质，让灯泡和灯管看起来更真实。

---

## 准备工作：阻止外部光线

![](img/ce8e809d1d9f3c9be1def9d8b7e69eb1_1.png)

![](img/ce8e809d1d9f3c9be1def9d8b7e69eb1_2.png)

上一节我们创建了室内场景，但当前场景缺少光照。在放置任何灯光之前，我们需要确保室外的阳光不会穿透墙壁照射到室内。

为了实现这一点，我们需要将场景中的几何体转换为静态网格体。静态网格体不会让光线穿透，而几何体则会。

以下是操作步骤：
1.  在“大纲视图”中，选中代表窗户和墙壁的所有几何体。
2.  在“细节”面板中，找到“创建静态网格体”选项。
3.  点击该选项，将其命名为“Studio”，然后点击“创建静态网格体”。

转换完成后，您会发现外部光线被阻挡了。为了在编辑时看得更清楚，我们可以暂时将视图模式从“光照”切换为“无光照”。

## 创建第一盏室内灯

现在，让我们开始创建第一盏室内灯。我们将在吧台下方放置一盏聚光灯。

首先，从内容浏览器中拖入一个灯具的静态网格体模型。请注意，这个模型本身不会发光，它只是一个外壳。

![](img/ce8e809d1d9f3c9be1def9d8b7e69eb1_10.png)

![](img/ce8e809d1d9f3c9be1def9d8b7e69eb1_11.png)

接下来，我们需要为其添加一个光源：
1.  点击顶部菜单的“添加” -> “光源”。
2.  选择“聚光灯”，将其拖放到场景中灯具模型的下方。
3.  调整聚光灯的位置和角度，使其看起来像是从灯具中照射出来的。

此时，由于我们处于“无光照”模式，还看不到灯光效果。让我们将视图模式切换回“光照”，就能看到灯光投射在地板和吧台上的效果了。

## 调整场景曝光

场景可能仍然很暗。为了整体控制场景亮度，我们需要添加一个后期处理体积。

![](img/ce8e809d1d9f3c9be1def9d8b7e69eb1_16.png)

![](img/ce8e809d1d9f3c9be1def9d8b7e69eb1_17.png)

操作步骤如下：
1.  点击“添加” -> “视觉效果” -> “后期处理体积”。
2.  将其拖入场景，并在“细节”面板中勾选“无限范围（未绑定）”，使其影响整个场景。
3.  在后期处理体积的设置中，找到“曝光”选项。
4.  将“测光模式”设置为“手动”，然后调整“曝光补偿”值（例如设为20），直到场景亮度合适。

## 创建自发光灯泡

![](img/ce8e809d1d9f3c9be1def9d8b7e69eb1_19.png)

![](img/ce8e809d1d9f3c9be1def9d8b7e69eb1_20.png)

现在灯光能照亮环境了，但如果我们看向灯具内部，会发现里面是空的，没有灯泡。这看起来不真实。我们需要创建一个能自发光的灯泡。

首先，添加一个球体作为灯泡模型，并调整其大小，放入灯具内部。

![](img/ce8e809d1d9f3c9be1def9d8b7e69eb1_22.png)

这个球体目前只是一个普通模型，不会发光。我们需要为它创建一个能发光的材质：
1.  在内容浏览器中右键点击，选择“材质” -> “材质”，将其命名为“M_Light”。
2.  将这个新材质拖放到球体“细节”面板的“材质”槽中。
3.  双击打开“M_Light”材质进行编辑。

在材质编辑器中，我们需要让材质自发光：
1.  找到“自发光颜色”输入引脚。
2.  在空白处右键点击，在搜索栏中输入“Constant3Vector”（三维常量），这是一个可以定义颜色的节点。
3.  将该节点拖到“自发光颜色”引脚上连接起来。
4.  双击“Constant3Vector”节点，选择白色（或任何你想要的发光颜色）。
5.  保存材质。

现在，球体就变成了一个会发光的灯泡。请注意，这种材质发光主要用于视觉效果，真正照亮场景的还是我们之前创建的聚光灯。

为了管理方便，可以将灯泡模型和聚光灯都拖放到灯具静态网格体下，使其成为灯具的子级。这样移动灯具时，所有部件会一起移动。

## 使用矩形灯

除了聚光灯，室内也常用到条形灯（如日光灯管）。对于这种灯具，使用“矩形灯”更为合适。

以下是创建条形灯的步骤：
1.  从资源库中拖入一个长条形的灯具静态网格体（如荧光灯天花板）。
2.  点击“添加” -> “光源” -> “矩形灯”，将其放入灯具模型内。
3.  调整矩形灯的位置、旋转、宽度和高度，使其与灯具模型匹配。
4.  您还可以调整“挡光板角度”等参数，控制光线的散射范围。

![](img/ce8e809d1d9f3c9be1def9d8b7e69eb1_27.png)

同样地，我们也需要创建发光的灯管模型：
1.  添加一个“圆柱体”，调整其大小和旋转，放入灯具内作为灯管。可以放置多个。
2.  将之前创建的“M_Light”材质赋予这些圆柱体。
3.  最后，将矩形灯和所有灯管模型都设为灯具静态网格体的子级，以便统一管理。

## 灯光属性详解

选中任何一盏灯，在“细节”面板中可以调整其属性以更好地融入场景：
*   **强度**：控制灯光的亮度。
*   **光源颜色/色温**：可以设置具体的颜色，或使用色温（如暖黄或冷白）来模拟真实世界的光线。
*   **间接光照强度**：控制光线在场景中反弹（间接照明）的强度。
*   **体积散射强度**：控制这盏灯与场景中的雾效（如体积雾）相互作用产生的“上帝之光”效果的强度。您可以单独为每盏灯调整此值，而不必全局提高雾的密度。

## 总结

本节课中，我们一起学习了如何为虚幻引擎中的室内场景设置照明。

![](img/ce8e809d1d9f3c9be1def9d8b7e69eb1_38.png)

我们首先通过将几何体转换为静态网格体来阻挡外部光线。然后，我们学会了放置和配置**聚光灯**与**矩形灯**这两种常用的室内光源，并通过后期处理体积控制了整体曝光。最后，我们创建了一个简单的**自发光材质**，并利用**Constant3Vector**节点为其赋予颜色，让灯泡和灯管模型看起来更真实。

![](img/ce8e809d1d9f3c9be1def9d8b7e69eb1_40.png)

记住，良好的照明是营造场景氛围的关键。接下来，我们可以继续在场景中放置更多灯光，并在后续课程中通过后期处理体积和雾效来进一步微调，使场景更具电影感。
