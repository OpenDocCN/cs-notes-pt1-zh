# 004：不同跟踪方法 🧭

![](img/841e2e1f69687d382f869b2e3dd594ab_0.png)

在本节课程中，我们将学习并比较几种常见的项目跟踪方法。理解这些工具将帮助你选择最适合你项目需求的跟踪方式，确保项目按计划推进。

上一节我们强调了跟踪的重要性，本节中我们来看看几种具体的跟踪方法。

你的项目计划旨在指导项目执行，因此它至少会包含一种跟踪方法。有时，根据团队需求，你可能会使用不止一种。本视频将重点讨论**甘特图**、**路线图**和**燃尽图**。选择哪种类型取决于你认为哪种最适合你的项目。重要的是选择整个团队都能轻松理解、参考并及时更新的方法。

以下是三种常见跟踪方法的详细比较：

**甘特图**或许是最常见的跟踪方法。它是一个久经考验的、能确保项目按计划进行的工具。甘特图将任务与时间进行对比，并包含诸如任务负责人和任务顺序等有用信息。因此，它对于保持进度以及管理具有许多依赖关系的项目非常有用。它同样适用于人员较多的团队，因为随着时间顺序推进，任务的所有权和责任被清晰地可视化呈现。这看起来像瀑布，因此甘特图常用于瀑布式项目管理。

在甘特图中：
*   每个任务由一个水平进度条表示。
*   进度条的长度取决于分配给该任务的时间量。
*   进度条上下堆叠，表示顶部的任务必须在下面的任务开始前完成。

甘特图通常存在于你的项目计划中，并随着项目进展而更新。

**路线图**在谷歌内部是另一种常见的跟踪方法。当你需要一种方式来跟踪项目中的重大里程碑时，路线图最为适用。它有助于向团队和关键利益相关者说明项目应如何随时间推进。

一个路线图示例可能如下：
*   顶部列出项目目标以及为实现这些目标将采取的方法描述。
*   包含一个**高层级项目概述**，即用三到四句话清晰陈述项目目标和优先事项的简明摘要。
*   目标、方法和概述下方是一个表格，勾勒出流程将包含的内容。表格可能按季度划分，反映项目时间线。
*   每个季度下列出关键里程碑，以及每位团队成员或部门需要完成的任务。大多数任务可能映射到同一季度内到期的里程碑。

路线图同时跟踪个人和项目在达成里程碑方面的进展。

**燃尽图**是三种跟踪方法中最精细的一种。燃尽图将时间与已完成的工作量和剩余工作量进行对比。其主要用途是让项目团队掌握目标完成日期，并在范围蔓延发生时及时察觉。燃尽图最适合需要对项目关联的每个任务进行详细分解审查的项目，以及那些按时完成是首要任务的项目。

在燃尽图中：
*   **Y轴**（垂直轴）象征待完成的任务数量。
*   **X轴**（水平轴）代表时间。
*   进度从图表的左上角开始跟踪。随着项目进行，你将向下（朝着剩余任务数为0的方向）和向右（朝着结束日期的方向）推进。
*   通常有一条**虚线**表示基于团队预期任务完成速度的预期进度，以及一条**实线**代表实际进度。

在继续推进项目之前，你需要决定哪种跟踪器最合理。
*   如果需要向大型团队传达里程碑，可以选择路线图。
*   如果项目具有多重依赖关系，可以选择甘特图。
*   如果对照截止日期跟踪任务尤为重要，那么燃尽图可能是最佳选择。
*   你甚至可能决定使用不止一种方法。

![](img/841e2e1f69687d382f869b2e3dd594ab_2.png)

因为跟踪方法将由你正在进行的项目类型、资源和项目范围决定。在你的职业生涯中，你很可能会在某个阶段使用多种类型的跟踪方法。例如，在项目开始时使用甘特图来规划工作范围，然后在发布前几周切换到燃尽图以确保一切就绪。
