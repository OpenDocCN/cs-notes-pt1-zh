在本节课中，我们将要学习Prometheus查询语言中的二元操作符。二元操作符是使用两个操作数的运算符，例如加法运算符和相等运算符。这些运算符不仅限于简单的算术运算，还能应用于瞬时向量，使得我们的查询语言变得更加强大和有趣。

![](img/7a322c76a484e906d24b1d242c6feae1_6.png)

## 概述

![](img/7a322c76a484e906d24b1d242c6feae1_8.png)

上一节我们介绍了PromQL的基本查询。本节中我们来看看二元操作符，它允许我们对标量和瞬时向量进行各种运算，包括算术、比较和逻辑操作，从而进行更复杂的数据分析和过滤。

![](img/7a322c76a484e906d24b1d242c6feae1_9.png)

## 标量与瞬时向量

![](img/7a322c76a484e906d24b1d242c6feae1_13.png)

除了运算符，我们还需要理解操作数的类型。瞬时向量是带有时间戳的数据点序列，而标量是简单的数值。

![](img/7a322c76a484e906d24b1d242c6feae1_15.png)

标量是可以写成数字后跟测量单位的形式的量。例如，1米是`1m`，1分钟是`1min`，200克是`200g`，5摄氏度是`5°C`。所有这些我们使用的测量值都称为标量。

![](img/7a322c76a484e906d24b1d242c6feae1_17.png)

![](img/7a322c76a484e906d24b1d242c6feae1_19.png)

我们可以在算术运算中将标量作为瞬时向量的操作数，从而改变瞬时向量中的值。

![](img/7a322c76a484e906d24b1d242c6feae1_21.png)

![](img/7a322c76a484e906d24b1d242c6feae1_22.png)

![](img/7a322c76a484e906d24b1d242c6feae1_24.png)

以下是使用标量修改瞬时向量的一个例子：

![](img/7a322c76a484e906d24b1d242c6feae1_26.png)

```promql
process_resident_memory_bytes / 1024
```

![](img/7a322c76a484e906d24b1d242c6feae1_28.png)

![](img/7a322c76a484e906d24b1d242c6feae1_30.png)

![](img/7a322c76a484e906d24b1d242c6feae1_31.png)

这个查询将`process_resident_memory_bytes`指标（显示内存大小）除以1024。结果是我们将以千字节为单位查看进程的内存使用情况，这使得图表上的数值变得更小、更易读。这个除法运算被应用于Prometheus中该时间序列的所有数据点。

![](img/7a322c76a484e906d24b1d242c6feae1_33.png)

## 算术运算符

![](img/7a322c76a484e906d24b1d242c6feae1_35.png)

![](img/7a322c76a484e906d24b1d242c6feae1_37.png)

![](img/7a322c76a484e906d24b1d242c6feae1_39.png)

二元算术运算符允许我们对数据进行数学计算。当两个操作数都是标量时，这非常有用，至少可以使我们的读数更容易理解。

![](img/7a322c76a484e906d24b1d242c6feae1_41.png)

![](img/7a322c76a484e906d24b1d242c6feae1_43.png)

以下是主要的算术运算符列表：

![](img/7a322c76a484e906d24b1d242c6feae1_45.png)

*   **加法**：`+`
*   **减法**：`-`
*   **乘法**：`*`（星号）
*   **除法**：`/`（斜杠）
*   **取模**：`%`（百分号）
*   **幂运算**：`^`

![](img/7a322c76a484e906d24b1d242c6feae1_46.png)

取模运算符可以返回一些非整数结果，如果你提供非整数输入。例如，`5 % 1.5`的计算结果是一个标量状态。

我们也可以将标量操作数放在运算符的左侧，将瞬时向量放在右侧。例如：

```promql
1e9 - process_resident_memory_bytes
```

![](img/7a322c76a484e906d24b1d242c6feae1_48.png)

![](img/7a322c76a484e906d24b1d242c6feae1_49.png)

这个查询会从10亿（1e9）中减去内存值。

![](img/7a322c76a484e906d24b1d242c6feae1_51.png)

## 比较运算符

比较运算符用于对值进行测试和过滤。我们已经在上一节课中看到了一些。

![](img/7a322c76a484e906d24b1d242c6feae1_53.png)

![](img/7a322c76a484e906d24b1d242c6feae1_55.png)

以下是主要的比较运算符列表：

![](img/7a322c76a484e906d24b1d242c6feae1_57.png)

*   **等于**：`==`
*   **不等于**：`!=`
*   **大于**：`>`
*   **小于**：`<`
*   **大于等于**：`>=`
*   **小于等于**：`<=`

我们可以在瞬时向量和标量之间使用比较。例如，我们可以过滤出值大于10的进程：

![](img/7a322c76a484e906d24b1d242c6feae1_59.png)

![](img/7a322c76a484e906d24b1d242c6feae1_61.png)

```promql
processes > 10
```

这个查询会返回值大于10的进程（例如，值为55的Prometheus进程）。而`processes < 10`则会返回值小于10的进程（例如，Node进程）。通过这种方式，我们可以进行数据过滤，例如，查找值大于10且小于某个值的进程。

![](img/7a322c76a484e906d24b1d242c6feae1_63.png)

![](img/7a322c76a484e906d24b1d242c6feae1_65.png)

## 逻辑运算符

![](img/7a322c76a484e906d24b1d242c6feae1_67.png)

![](img/7a322c76a484e906d24b1d242c6feae1_69.png)

逻辑运算符允许我们组合多个条件进行更复杂的查询。虽然视频中没有详细展开，但它们是二元操作符的重要组成部分。

![](img/7a322c76a484e906d24b1d242c6feae1_70.png)

以下是主要的逻辑运算符列表：

![](img/7a322c76a484e906d24b1d242c6feae1_72.png)

![](img/7a322c76a484e906d24b1d242c6feae1_73.png)

*   **且**：`and`
*   **或**：`or`
*   **除非**：`unless`

例如，`metric1 > 10 and metric2 < 5` 会筛选出同时满足两个条件的时间序列。

## 总结

![](img/7a322c76a484e906d24b1d242c6feae1_75.png)

本节课中我们一起学习了PromQL中的二元操作符。我们了解了如何对标量和瞬时向量使用算术运算符（如`+`, `-`, `*`, `/`）来转换数据单位，使其更易读。我们还探讨了比较运算符（如`>`, `<`, `==`），它们能帮助我们过滤出符合特定条件的时间序列。最后，我们简要提及了逻辑运算符（`and`, `or`, `unless`）用于组合查询条件。这些工具是进行有效指标分析的基础，使得Prometheus的查询语言功能强大且灵活。在接下来的课程中，我们将继续深入更多高级用法。
