# 088：CRUD命令详解 🗄️

在本节课中，我们将学习MongoDB中最基础、最重要的CRUD命令。CRUD代表创建、读取、更新和删除，是任何数据库管理员都必须掌握的核心操作。

## 概述

CRUD命令是用于创建、读取、更新和删除数据库内容的命令。在Linux环境下操作MongoDB会更为便捷。本节我们将从创建命令开始，逐步讲解每一个操作。

## 创建文档

上一节我们介绍了CRUD的概念，本节中我们来看看如何创建数据。在MongoDB中，我们使用`insert`命令向集合中插入文档。

例如，假设我们想在一个博客数据库中存储一篇帖子。首先，我们需要创建一个局部变量来存放文档内容，然后将其插入到集合中。

以下是创建并插入一篇博客帖子的步骤：

1.  创建一个名为`post`的变量，其内容是一个包含标题、内容和日期的文档。
    ```javascript
    post = {
        title: "My Blog",
        content: "my first post",
        date: new Date()
    }
    ```
    MongoDB使用类似JSON的键值对结构。`new Date()`函数会自动获取服务器当前的日期和时间。

2.  仅创建变量并不会将数据保存到数据库。需要使用`insert`命令将变量内容插入到指定的集合中。
    ```javascript
    db.blog.insert(post)
    ```
    命令执行后会返回插入结果，显示插入了多少条文档。

3.  使用`find`命令可以验证数据是否已成功插入。
    ```javascript
    db.blog.find()
    ```
    该命令会显示集合中的所有文档，包括自动生成的`_id`（主键）、标题、内容和日期。

## 读取文档

创建数据后，我们常常需要读取它们。MongoDB提供了`find`和`findOne`命令用于查询。

`find()`命令会返回集合中的所有文档。如果只想查看集合中的单个文档，可以使用`findOne()`命令，它能以更清晰、更有组织的方式显示键值对内容。

```javascript
db.blog.findOne()
```

## 更新文档

如果我们想对已存在的文档进行修改，例如为博客帖子添加一个评论字段，就需要使用更新操作。

以下是更新文档的步骤：

1.  使用`update`命令。我们需要指定查询条件（例如根据标题）和要更新的内容。
    ```javascript
    db.blog.update(
        { title: "My Blog" },
        { $set: { comments: [] } }
    )
    ```
    此命令会找到标题为“My Blog”的文档，并为其设置一个名为`comments`的空数组字段。

2.  更新后，可以再次使用`find`或`findOne`命令来确认更新是否成功。
    ```javascript
    db.blog.findOne()
    ```
    现在可以看到文档中多了一个`comments`字段。

## 删除文档

最后，我们来看如何删除数据。使用`remove`命令可以删除集合中的文档。

例如，要删除所有标题为“My Blog”的文档，可以执行：

```javascript
db.blog.remove({ title: "My Blog" })
```

执行后，使用`findOne()`查询会返回`null`，表明相关数据已被完全删除。

**请注意**：这是一个不可逆的操作。如果不提供任何参数，`remove({})`命令会删除集合中的**所有**文档，使用时需格外谨慎。

## 总结

本节课中我们一起学习了MongoDB的CRUD基本操作：使用`insert`创建文档，使用`find`和`findOne`读取文档，使用`update`更新文档，以及使用`remove`删除文档。这些是操作MongoDB的基石，我们将在后续课程中更深入地应用它们。
