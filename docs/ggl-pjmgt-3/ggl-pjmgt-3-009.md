# 009：项目规划阶段总结 🎯

![](img/94afbfbc0d1ab5cdaa52680b11689463_0.png)

在本节课中，我们将回顾项目生命周期中规划阶段的核心内容。规划阶段是项目成功的关键，它涉及制定时间表、设定预算以及准备风险管理计划。

## 规划阶段的核心要素

上一节我们介绍了如何启动项目生命周期的规划阶段。本节中，我们来总结该阶段涵盖的主要活动和产出。

以下是规划阶段需要完成的关键工作：
*   制定项目进度表。
*   设定项目预算。
*   准备风险管理计划。

## 项目启动会的重要性

规划阶段的一个重要组成部分是项目启动会。这是一个关键节点，你可以与团队和利益相关者会面，以达成共同愿景。

以下是项目启动会的主要目标：
*   就项目范围达成一致。
*   建立团队融洽关系。
*   提出问题并澄清疑问。
*   设定明确的期望。

## 里程碑与任务

我们还讨论了**里程碑**，它们是项目时间表中标志进展的重要节点。同时，我们学习了**任务**，它们是在特定时间段内需要完成的具体活动。

你了解了里程碑的价值以及如何设定它们。接着，你学会了如何将这些里程碑分解为具体的任务，并将这些信息记录在**工作分解结构**中。
