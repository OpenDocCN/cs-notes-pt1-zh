# 029：获取采购支持 🛒

![](img/0108bdb539cc54bab0ed6787047125b8_0.png)

在本节中，我们将探讨在采购流程中如何获取其他部门和团队成员的支持。采购流程并非在选择供应商和签署合同后就结束了，项目经理需要持续跟踪供应商表现，并确保所有合同与操作符合法律及公司合规要求。

上一节我们介绍了保密协议、征求建议书和工作说明书等采购文件。本节中我们来看看如何在整个采购周期中协同法律、合规等部门，确保项目平稳运行。

## 跟踪与评估供应商表现

签署合同后，你需要使用绩效跟踪工具和定期会议来监督供应商表现。

以下是两种常见的跟踪与评估方法：

*   **绩效跟踪器**：用于量化记录供应商在关键指标上的表现。
*   **季度业务评审**：定期举行的会议，旨在评估供应商的整体表现，确保双方履行最初的协议。

## 获取法律团队的支持

当合同中包含你不熟悉的专业术语时，你需要寻求法律团队的帮助。公司的法律支持情况因其规模而异。

以下是几种常见的公司法律支持架构：

*   **内部法律团队**：通常存在于成熟的大型企业。
*   **外包法律团队**：一些公司选择将法律事务外包。
*   **外部法律顾问**：初创公司或小型企业可能没有常设法律团队，但会在需要时咨询法律顾问。

## 理解合规要求

不同国家的产品制造和消费者权益相关法律各不相同。但几乎所有地区都有关于公平与道德贸易的法律需要遵守。大多数公司都有自己的使命或价值观声明。

合规部门的职责是确保日常运营符合公司价值观和政府政策。

以下是合规团队的一些核心职责：

*   确保公司运营符合其价值观声明与政府政策。
*   致力于防止歧视行为。
*   确保公司履行足够的企业社会责任。

作为项目经理，你需要确保代表公司的各利益相关方也遵守这些政策和流程。你必须了解所有与法律或合规问题相关的重要会议，并提醒团队按时参加。

## 确保关键审批与审查

![](img/0108bdb539cc54bab0ed6787047125b8_2.png)

在需要某种必要批准时，让适当的利益相关方参与进来尤为关键，因为这将以某种方式影响项目。与大多数项目一样，对于“办公室绿植”项目，让法律和合规部门参与审查和批准非常重要。

例如，假设你配送植物的部分办公室允许携带宠物。在与供应商的合同中，你明确要求只提供无毒植物。但如果供应商错误地提供了一种对动物有害的植物，由于遗漏了这一关键细节，“办公室绿植”公司可能会承担法律责任并陷入法律纠纷。这是一个非常关键的考量。

因此，无论你未来公司的法律团队或顾问情况如何，以某种形式对合同进行审查都具有良好的商业意义，并应作为项目经理的优先事项。

## 总结与预告

本节课中我们一起学习了在采购流程中获取支持的重要性。我们了解到，项目经理需要持续跟踪供应商绩效，并主动协同法律与合规团队审查合同与操作，以规避风险，确保项目符合所有规定。
